# Numinas

Official marketing site for **Numinas** — a creative studio for scroll-stopping motion that clarifies and converts.

## About

Numinas builds visuals that travel across every channel: launch films, motion brand systems, social loops, explainers, and full production from concept to delivery.

**Studios:** Vancouver · Toronto

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- GSAP (pillars / interactions)
- Motion (UI components)

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

Output: `dist/` — ready for Vercel or any static host.

## Media assets

| Path | Purpose |
|------|---------|
| `public/media/hero/demoreel/` | Hero background reel |
| `public/media/pillars/` | Pillar hover visuals |
| `public/fonts/Mori/` | PP Mori (Book, Regular) |
| `public/fonts/Hatton/web/` | PP Hatton (Medium, Medium Italic) |

## Deploy (Vercel)

1. Push this repo to GitHub
2. Import in [Vercel](https://vercel.com) → Framework: **Vite**
3. Build: `npm run build` · Output: `dist`
